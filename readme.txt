== Description ==
SSVP can be used for visualizing site structure in graphial format. Helps displaying custom post types,taxonomies and terms in hierarchical format.

== Installation ==
1.Just download plugin > Upload plugin > Upload/Install plugin > Activate 
2.Go to Plugin Setting Page for available options.
